# MInecraft-bot 🤖

Un bot de Minecraft que juega de forma automática en servidores, comportándose como un jugador normal que se conecta sin launcher.

## Descripción

Este proyecto implementa un bot que puede:
- Conectarse a servidores de Minecraft como un cliente normal
- Ejecutar acciones automáticas (movimiento, construcción, recolección, etc.)
- Interactuar con el entorno del servidor
- Comportarse de manera similar a un jugador real

## Características

- 🎮 Conexión a servidores Minecraft estándar
- 🤖 Automatización de tareas
- 👤 Comportamiento de jugador normal
- 🔗 Sin dependencia de launcher
